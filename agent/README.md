### AI Agent

An AI agent is an intelligent system that receive a high level goal from a user and autonomously 
plans, decides and executes a sequence of actions by using external tools, APIs or Knowledge sources
all while maintaining context, reasoning over multiple steps adapting to new information and optimizing
for intended output.

### AI Agent Characteristics 
1. **Goal Driven** : you tell agent what u want, not how to do it.
2. **Autonomous planning** : Agent break down the problem and sequences tasks on its own.
3. **Tools** : Agents know what tools it has, and when to use them.
4. **Context Aware** : Maintains memory across all steps to inform future actions
5. **Adaptive** : Rethink plan when things change like in between goal changes. 

### Agent Pattens

Agentic patterns are design patterns for building AI agents:
1. ReAct Pattern - Reason + Act
2. Plan-and-Execute Pattern - Split agent : Planner and Executor
3. Reflection Pattern (Self-Critique)
4. Multi-Agent Pattern : Multi agent collaboration
5. Iterative
6. RAG Agent 

.. and so on.
